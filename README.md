# homebrew-caffbar

Homebrew tap for installing **CaffBar**.

## Install

```bash
brew install --cask akalp/caffbar/caffbar
```

Or:

```bash
brew tap akalp/caffbar
brew install --cask caffbar
```

## Updates

This tap is intended to be updated automatically by the `caffbar` release workflow.
If automation is not configured yet, update `Casks/caffbar.rb` manually with the new version and SHA-256.
